*******COMANDI BASE*******

Creare una virtual environment: --> python -m venv venv

Sbloccare le policy (Windows): --> Set-ExecutionPolicy -Scope Process -ExecutionPolicy Bypass

Attivare la virtualenv nello stesso percorso dove stai lavorando il progetto: --> ./venv/Scripts/Activate.ps1

Per disattivare: deactivate

Installare pacchetti da requirements.txt: --> pip install -r requirements.txt

Salvare i pacchetti installati in requirements.txt --> pip freeze > requirements.txt
